O que é um evento?

Um evento é uma ação reconhecida pelo navegador ou pelo ambiente de desenvolvimento que desencadeia algum tipo de resposta no código. Por exemplo, clique do mouse, pressionamento de tecla, movimento do mouse, etc.
Qual é a diferença entre um evento de elemento HTML e um evento React?

Os eventos em elementos HTML são tratados diretamente pelo DOM, enquanto no React, os eventos são sintéticos, criados pelo React para padronização entre navegadores. Em termos práticos, o manuseio de eventos em React segue uma abordagem mais uniforme, independente do navegador.
Escreva pelo menos 4 eventos de teclado:

javascript

document.addEventListener('keydown', (event) => {
  console.log('Key down:', event.key);
});

document.addEventListener('keyup', (event) => {
  console.log('Key up:', event.key);
});

document.addEventListener('keypress', (event) => {
  console.log('Key pressed:', event.key);
});

document.addEventListener('input', (event) => {
  console.log('Input:', event.target.value);
});

Quais são os eventos de teclado e mouse mais comuns?

Os eventos de teclado mais comuns são keydown, keyup, keypress, input, change e blur. Os eventos de mouse comuns incluem click, dblclick, mouseenter, mouseleave, mousedown, mouseup, mousemove e wheel.

Qual é a diferença entre onInput, onChange e onBlur?

    onInput: Dispara imediatamente quando o valor de um elemento de entrada é alterado.
    onChange: Dispara quando um valor de um elemento de entrada é alterado e o foco é perdido do elemento.
    onBlur: Dispara quando o elemento perde o foco.

Onde colocamos o evento onSubmit?

O evento onSubmit é usado em elementos de formulário (<form>) e geralmente é adicionado como um atributo no elemento <form> ou é atribuído através de manipulação de eventos no JavaScript. Ele é acionado quando o formulário é enviado, seja pelo pressionamento do botão de envio ou através de JavaScript.
